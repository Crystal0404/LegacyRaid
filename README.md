# LegacyRaid

**中文 / [English](https://github.com/Crystal0404/LegacyRaid/blob/master/README_EN_US.md)**

### 介绍

*你是旧版袭击的粉丝吗?*

这是一个简单的mod, 它把1.20.x的袭击机制移植到了1.21+

它不需要额外依赖项

### 多加载器支持

| 加载器      | 是否支持 |
|----------|------|
| Fabric   | √    |
| NeoForge | √    |
| Quilt    | √    |
| Forge    | X    |

**注意: 我不会为Forge加载器添加支持, Quilt加载器用户下载Fabric版本即可**
